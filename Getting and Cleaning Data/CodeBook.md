# Getting and Cleaning Data Course Project

## Code Book
The source of this dataset comes from the "Human Activity Recognition Using Smartphones Dataset".


Each row of the dataset contains the "Subject" and "Activity" variables and 66 variables as described in the original dataset.
**"Subject"** is a numerical value from 1 to 30 referencing a tester.
**"Activity"** is one string of WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING.
The 66 others are the average of the values found in the original dataset for each subject / activity.
Their names are unchanged, there is no unit: the original values where normalized [-1, +1] without unit.
